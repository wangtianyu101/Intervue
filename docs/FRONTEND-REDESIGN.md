# CodeMock 前端重设计方案 V1.0

> 现状：登录仅 GitHub OAuth，进入后直接跳面试配置页，信息架构扁平。
> 目标：账号密码登录 + 三区仪表盘 + 面试历史 + 能力分析系统。

### 6.2 分析数据来源

| 指标 | 计算方式 |
|---|---|
| **综合得分** | 所有面试 `overall_score` 加权平均（最近面试权重更高） |
| **雷达图** | 按 topic 聚合所有 `question_records.score`，取均值 |
| **趋势** | 同 topic 不同时期的得分变化斜率 |
| **薄弱项** | 得分最低的 3-5 个 topic，且出现频率 > 1 |
| **推荐练习** | 薄弱项 ∩ 高频考点，按 (薄弱程度 × 考试频率) 排序 |
